## Usage

```js
import Pinchable from 'react-native-pinchable';

<Pinchable>
  <Image source={...}>
</Pinchable>
```

### Properties

| Prop                   | Description                     | Default |
| ---------------------- | ------------------------------- | ------- |
| **`minimumZoomScale`** | The minimum allowed zoom scale. | `1`     |
| **`maximumZoomScale`** | The maximum allowed zoom scale. | `3`     |
